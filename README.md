# AI Image Generator


## Features

- Streamlit frontend
- Image prompt input
- Style selection
- Hugging Face API integration
- Prompt history
- Download generated images


## Installation

pip install -r requirements.txt


## Run

streamlit run app.py


## API Key

Create a .env file


HF_TOKEN=your_token_here


## Deployment

Deploy on Streamlit Community Cloud


## Limitation

Free Hugging Face credits are limited.
